Quiz

  • What is the difference between var and let?
  • Answer: Var can be used to reassign and redeclare, but let can't be used to redeclare. Let creates a block scope while var can be used to access a hoisted variable.

  • What is the difference between var and const?
  • Answer: Var can be used to reassign and redeclare, but const can't be used to reassign and redeclare. Var can be used to access a hoisted variable while const creates block scope.

  • What is the difference between let and const?
  • Answer: Let can be used to reassign but const can't be used to reassign or redeclare.

  • What is hoisting?
  • Answer: Hoisting refers to the process where the interpreter to move the declaration to the top of their scope before execution of the code.
